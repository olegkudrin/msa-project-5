# План реализации решения

## Подготовка инфраструктуры

| Действие                       | Описание                                                          |
| ------------------------------ | ----------------------------------------------------------------- |
| Настройка рабочей базы         | Убедиться, что база доступна, есть необходимые таблицы и индексы. |
| Настройка объектного хранилища | Развернуть MinIO, создать bucket для прайс-листов.                |
| Настройка Kubernetes           | Подготовить namespace, service account, secrets.                  |

## Разработка Spring Batch приложения

| Действие                       | Описание                                                          |
| ------------------------------ | ----------------------------------------------------------------- |
| Создание проекта               | Spring Boot: Spring Batch, PostgreSQL Driver, AWS SDK, Apache POI |
| Модель данных                  | Создать entity классы: Product, Category, Client, ClientPrice.    |
| Компонент Reader               | Реализовать SQL запрос к таблицам для чтения исходных данных.     |
| Компонент Processor            | Подготовка данных для прайсов (группировка по клиентам).          |
| Компонент Writer               | Генерация прайсов в виде csv/xlsx файлов по клиентам.             |
| Конфигурация Job               | Создать Job с шагами: read data, process data, generate files.    |
| Конфигурация расписания        | Подготовить приложение для внешнего запуска.                      |

## Конфигурация и деплой

| Действие                       | Описание                                                          |
| ------------------------------ | ----------------------------------------------------------------- |
| Application properties         | Настроить доступ к базе, хранилищу и сервису уведомлений.         |
| Docker образ                   | Создать Dockerfile для Spring Boot приложения.                    |
| Kubernetes манифесты           | Создать: CronJob, ConfigMap, Secret.                              |
| CI/CD пайплайн                 | Настроить автоматическую сборку и деплой.                         |

## Мониторинг и логирование

| Действие                       | Описание                                                          |
| ------------------------------ | ----------------------------------------------------------------- |
| Логирование                    | Настроить логирование исполнения Spring Batch Job.                |
| Метрики                        | Добавить метрики: кол-во записей, файлов, время работы, ошибки.   |
| Алерты                         | Настроить алерты на ошибки выполнения Job.                        |

## Тестирование

| Действие                       | Описание                                                          |
| ------------------------------ | ----------------------------------------------------------------- |
| Unit тесты                     | Написать тесты для Reader, Processor, Writer.                     |
| Integration тесты              | Протестровать Job с тестовой БД.                                  |
| E2E тестирование               | Запустить Job в тестовое среде, проверить генерацию файлов.       |
